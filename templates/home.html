Most asked questions

How to use template?

You can easily select templates from the right drop down menu when you send email and mass email.

How to send mass email?

First of all you must have recent email sends or add people you want to send email. Then you can add people in group. You can filter them in criteria like preferences and age.Then you can send mass email

How to handle "Email password error"?

In order to send emails from your own you must authenticate yourself with your credentials. If you don't post your password in edit profile page or post wrong password you won't be able to send emails

How to edit group?

If you want to edit group you must go to add new group and just type the same name. This will replace the old one with new

How to edit receiver?

Same way as groups. You must just go to add receiver with the same name and will replace the old one
